# Add to Cart Functionality in JavaScript

A simple and responsive Add to Cart functionality built using HTML, CSS, and JavaScript. This project demonstrates how users can add products to a shopping cart dynamically with real-time cart updates.

## 🚀 Features

- Add products to cart
- Increase/decrease product quantity
- Remove items from cart
- Dynamic cart item count
- Real-time total price calculation
- Responsive UI design
- Beginner-friendly JavaScript project

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 📂 Project Structure

├── index.html
├── style.css
├── script.js
└── assets/
